# 💰 FinTrack – Full Stack Finance Tracker

FinTrack is a modern full-stack web application that helps users track income and expenses with a clean dashboard, fast interactions, and real-time updates.

---

## 🚀 Features

### 🔐 Authentication
- Register / Login
- JWT-based authentication
- Protected routes

### 💸 Transactions
- ➕ Add Transaction
- ✏️ Edit Transaction
- ❌ Delete Transaction
- 📂 Category selection (Income / Expense)
- 📅 Date-based tracking

### 📊 Dashboard
- Total Income
- Total Expenses
- Balance
- Monthly summary (client-side & API)
- Recent transactions

### 📋 Transactions Table
- 🔍 Search (title, category)
- 🎛 Filters (type, category)
- 🔃 Sorting (newest/oldest)
- 🧾 Clean table UI

### ⚡ Performance
- TanStack Query (React Query)
- Automatic data refresh with `invalidateQueries`
- Optimized rendering

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- TanStack Query (React Query)
- Axios
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
